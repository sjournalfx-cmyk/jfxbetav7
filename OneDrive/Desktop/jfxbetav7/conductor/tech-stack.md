# Tech Stack

## Frontend
- **Language:** TypeScript
- **Framework:** React 19
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Libraries:**
    - Framer Motion
    - Lightweight Charts
    - TipTap (Rich Text Editor)
    - Mermaid.js (Diagrams)

## Backend & Database
- **Platform:** Supabase
    - **Database:** PostgreSQL
    - **Features:** Authentication, Real-time Subscriptions

## AI Integration
- **Provider:** Google Gemini API

## Testing
- **Framework:** Vitest
