# AGENTS.md - JournalFX

## Commands

```bash
npm install          # Install dependencies
npm run dev          # Dev server on port 3000 (NOT 5173)
npm run build        # Production build → dist/
npm run test         # Vitest tests
npm run lint         # TypeScript typecheck (tsc --noEmit)
npm run supabase:migrate   # Create and push migration
npm run supabase:reset     # Reset database
```

## Environment Setup

Create `.env.local` with:
- `VITE_SUPABASE_URL` - Supabase project URL
- `VITE_SUPABASE_ANON_KEY` - Supabase anon key
- `VITE_GEMINI_API_KEY` - Google Gemini API key
- `VITE_NVIDIA_API_KEY` - NVIDIA API key (optional)

## Project Structure

- **components/** - UI components (Dashboard, Journal, Analytics, etc.)
- **services/** - API layer (dataService.ts, authService.ts, metaApiService.ts)
- **hooks/** - Custom hooks (useAuth.ts, useData.ts, useLocalStorage.ts)
- **lib/** - Utilities and constants
- **supabase/** - Migrations and edge functions
- **test/** - Vitest setup and tests

## Key Patterns

- Path alias `@/` maps to project root (configured in tsconfig.json)
- State: React hooks for local, Supabase for server state
- Real-time sync via Supabase channels (trades_sync, profile_sync, etc.)
- Heavy components (Analytics, AIChat, BacktestLab, Notes) lazy-loaded

## Testing

- Test setup: `test/setup.ts` imports jest-dom matchers
- Run single test: `npm test -- --run <test-file>`
- Environment: jsdom

## Build

- Output: `dist/` (generated by Vite)
- Dev server proxies: `/api/metaapi-*` → MetaAPI, `/api/nvidia` → NVIDIA API

## Important Notes

- Dev server runs on port **3000**, not default Vite 5173
- Tailwind CSS v4 (uses `@tailwindcss/postcss`, not v3 config)
- React 19 with strict mode enabled
- TypeScript strict mode with `noEmit: true`
