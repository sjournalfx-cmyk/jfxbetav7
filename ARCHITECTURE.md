# Architecture

## Overview
JournalFX is a single-page application (SPA) built with React 19 and TypeScript. It uses React hooks for local state management and Supabase for server-side state and real-time synchronization. The architecture emphasizes modularity, type safety, and scalability for trading journal features.

## Key Directories
- **components/**: UI modules (e.g., Dashboard.tsx for overviews, Journal.tsx for trade CRUD). Subfolders for analytics/, ui/ primitives.
- **services/**: API integrations (dataService.ts for Supabase CRUD, authService.ts for authentication, metaApiService.ts for external sync).
- **hooks/**: Custom hooks (useLocalStorage.ts for persistence).
- **lib/**: Utilities (constants.ts for tiers, Supabase client setup).
- **supabase/**: Database functions, migrations, edge functions.
- **public/**: Static assets (images, icons).
- **dist/**: Built output (generated by Vite).

## State Management
- **Local State**: React hooks (useState, useEffect) for component-level data (e.g., form inputs).
- **Server State**: Supabase handles persistent data with real-time subscriptions. No external state library; relies on React Query-like patterns via services.

## Data Flow
1. **Authentication**: authService.ts handles login → sets user context in App.tsx.
2. **Onboarding/Dashboard**: Loads user profile/goals via dataService.ts.
3. **CRUD Operations**: Components call services (e.g., addTrade in Journal.tsx) → Supabase API → real-time updates.
4. **Real-Time Sync**: Supabase channels (e.g., trades_sync) push changes to subscribed components.

## Real-Time Sync
- Uses Supabase real-time for live updates (trades, profiles, notes, goals).
- Channels: trades_sync, profile_sync, etc., subscribed in App.tsx.

## Technologies
- **Frontend**: React 19 + TypeScript (strict mode).
- **Build**: Vite (dev server, optimized builds).
- **Styling**: Tailwind CSS + custom themes.
- **Backend/Database**: Supabase (PostgreSQL + API, auth, storage).
- **AI**: Google Gemini API for insights.
- **UI Libraries**: Lucide React (icons), TipTap (rich editor), DND Kit (drag-drop), Motion (animations), Mermaid (diagrams), Canvas Confetti (effects).
- **Charts**: TradingView widget.
- **External**: MetaAPI for trade sync.
